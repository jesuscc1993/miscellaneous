Description

  • Allows the customization of the game's layout and responsive behavior.

Required game settings

  • Side Bar Type: Slide

Description

  • Imgur album

Installation

  • UserCSS

Customization settings

  1. (Center content) Submenu not accounted for (see album screenshot)
  2. (Center popups) Warning: might make some elements inaccessible depending on the device's size
  3. (Fit crew chat) Fits crew chat in the available space (requires Center content)
  4. (Fix header menu position) Keeps header menu in place when scrolling
    • Optional: hides contained banners (use when they won't fit the screen)
  5. (Sort summons) Sort summons by uncap level
  6. (Rearrange co-op room) Put room buttons on top
  7. (Rearrange raid UI) Move raid log to the bottom (requires +1080p or zooming out)
  8. (Compact summons) Compacts summon cards in the selection page (see album screenshot)
  9. (Compact lists) Show 5 items per row instead of 4 (grids end up being 5x4 instead of 4x5, saving some scrolling)
  10. (Hide crystal count) Hides crystal count from #mypage and #profile in an attempt to stop you from impulsively wasting instead of saving
  11. (Hide quest commentary) Hides Vyrn's commentary on the summon selection screen
